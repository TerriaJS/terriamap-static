# Static TerriaMap

This is a pre-built version of [TerriaMap](https://github.com/TerriaJS/terriamap).

To run this map, you need to have [Node.js](https://nodejs.org/) installed, then run the following command:

```bash
npx serve -p 3000
```

Then go to [http://localhost:3000](http://localhost:3000) to view the map.

## Edit configuration and catalog

Edit [`config.json`](./config.json) to change the configuration of the map.

Edit [`init/foss4g.json`](./init/foss4g.json) to change the catalog of the map.
